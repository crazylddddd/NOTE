# 文本对象

tag: #vim/文本对象

- 只能使用在`可视模式`或`操作符`后
- 以`a`开头则是选择一个包含空白的对象
- 以`i`开头则是不包含

| 符号                   | 作用                                         |
| ---------------------- | -------------------------------------------- |
| aw 或者[count]w        | 一个或者[count]个单词，包括单词之间的空白    |
| `note`:以下省略[count] |                                              |
| iw                     | 内含单词,包括单词之间的空白                  |
| aW                     | 一个字串，包括开头或结尾的空白，但不单独计算 |
| iW                     | 内含字串，字串之间的空格也算一个字串         |
